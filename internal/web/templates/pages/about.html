About

This is a Go web application template featuring:

  • Go + Chi — Lightweight HTTP router with middleware
  • PostgreSQL + pgx — Database with connection pooling
  • HTMX — Server-driven UI interactions
  • Alpine.js — Lightweight JavaScript reactivity
  • Tailwind CSS — Utility-first styling with dark mode
  • zerolog — Structured logging
  • golang-migrate — Database migrations
  • AWS CDK (Go) — Infrastructure as code
  • Docker Compose — Local development environment
